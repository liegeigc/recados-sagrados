# recados-sagrados

Deploy estatico (Vercel) das paginas publicas do produto **O Codigo do Amor Divino**.

Dominio: https://amor.sinalsagrado.online

## Estrutura

- `obrigada.html` — pagina pos-pagamento aprovado
- `aguardando.html` — pagina aguardando pagamento (boleto/pix/analise cartao)
- `img/` — assets estaticos (selo)

Atualizacoes futuras (pagina de vendas, etc) vao na raiz tambem.
